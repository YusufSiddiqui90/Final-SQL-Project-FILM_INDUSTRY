Project Overview

Phase-2 focuses on advanced SQL query writing. In this phase, different types of SQL queries are used to create, modify, manipulate, and retrieve data from the Film Industry database in a professional and structured way.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Query Languages & SQL Features
This phase includes practical use of all major SQL query languages and features:
DDL: CREATE, ALTER, DROP, TRUNCATE, RENAME
DML: INSERT, UPDATE, DELETE
DQL: SELECT, DISTINCT, aggregate functions
Queries are written using different operators, clauses, and aliases to perform meaningful data analysis and filtering.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Cascades, Clauses & Objective

Foreign key constraints with ON DELETE CASCADE and ON UPDATE CASCADE are implemented to maintain referential integrity. Queries also use WHERE, ORDER BY, GROUP BY, HAVING, and LIMIT clauses.
The main objective of Phase-2 is to demonstrate strong SQL query mastery, proper use of operators, clauses, aliases, and cascade rules with clean, well-commented queries.
