Project Overview

Phase-4 focuses on advanced database features to make the Film Industry database more efficient, secure, and closer to real-world business systems. This phase covers stored logic, automation, transaction control, and analytical queries.
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Advanced Database Features
In this phase, the following advanced SQL concepts are implemented:
Views (simple and complex views using joins and aggregates)
Cursors for row-by-row data processing
Stored Procedures for reusable business logic
Window Functions for analytical queries and ranking
DCL & TCL for access control and transaction management
Triggers for automatic actions and data integrity
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

Objective

The main objective of Phase-4 is to demonstrate advanced SQL skills by using views, procedures, cursors, window functions, transactions, security commands, and triggers. This phase helps in building a more professional, automated, and business-ready database system.
